## ✨ 앱 주요 기능
  > AI 스마트 카드 실적 매니저 Flutter 앱은 사용자의 카드 실적, 혜택, 지출 데이터를 한눈에 확인하고 <br>AI가 제안하는 최적 소비 전략을 실시간으로 받아볼 수 있는 스마트 금융 관리 앱입니다.<br>Flutter 기반의 크로스플랫폼 구조로 iOS와 Android에서 동일한 사용자 경험을 제공하며, AI 추천 UI, <br>실시간 실적 알림을 통해 가장 똑똑한 금융 비서를 구현합니다.
